# Required Tools & Installation

## Linux/macOS
- **file**: Built-in. Usage: `file image.jpg`
- **strings**: Built-in. Usage: `strings image.jpg`
- **exiftool**: `sudo apt install libimage-exiftool-perl` (or brew)
- **clamav**: `sudo apt install clamav` then `freshclam` for updates
- **imagemagick**: `sudo apt install imagemagick` for `convert`

## Python Libs (for scripts)
- Pillow: `pip install pillow`
- exifread: `pip install exifread`

## Online
- VirusTotal: Upload files at virustotal.com

## Full Workflow
Run in a VM for safety: VirtualBox + Ubuntu.
